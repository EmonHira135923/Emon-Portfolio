## 📖 Overview

A modern, responsive portfolio website built with **Next.js** and **Tailwind CSS**, featuring animated sections, ambient lighting effects, and a polished dark aesthetic. Designed to highlight expertise in full-stack development, competitive programming, and AI/ML.

---

## ✨ Features

| Feature                  | Description                                                      |
| ------------------------ | ---------------------------------------------------------------- |
| 🎬 **Hero Section**      | Animated typing effect cycling through developer roles           |
| 👤 **About Page**        | Background, education, and personal story                        |
| 🗂️ **Projects Showcase** | Full-stack, frontend, backend & competitive programming projects |
| 🛠️ **Skills Overview**   | Comprehensive tech stack with animated marquee display           |
| 📬 **Contact Form**      | Direct contact for collaborations and opportunities              |
| 📱 **Fully Responsive**  | Optimized layout across all device sizes                         |
| 🌑 **Dark Theme**        | Modern dark UI with gradient accents and ambient glow effects    |

## 🚀 Getting Started

### Prerequisites

- Node.js `v18+`
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/EmonHira135923/Emon-Portfolio.git

# 2. Navigate to the project directory
cd emon-portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser. 🎉

---

## 📦 Build & Deployment

```bash
# Build for production
npm run build

# Start the production server
npm start
```

Deploy instantly with [Vercel](https://vercel.com) — just connect your GitHub repo.
