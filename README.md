A modern, responsive portfolio website showcasing the work and skills of Emon Hossain Hira, a Full Stack Developer specializing in React, Next.js, and modern web technologies.

## 📋 Description

This portfolio website is built with Next.js and features a sleek, dark-themed design with interactive elements. It highlights Emon's expertise in full-stack development, including projects in web development, competitive programming, and AI/ML. The site includes sections for showcasing skills, projects, about information, and contact details.

## ✨ Features

- **Hero Section**: Animated typing effect displaying various roles (Full Stack Developer, React Developer, Next.js Engineer, etc.)
- **About Page**: Detailed information about Emon Hossain Hira, including background as a Diploma Computer Science student
- **Projects Section**: Showcase of full-stack, frontend, backend, and competitive programming projects
- **Skills Overview**: Comprehensive list of technical skills including C, C++, Python, JavaScript, React, Next.js, Node.js, MongoDB, and more
- **Skills Marquee**: Animated display of proficiency areas
- **Contact Form**: Easy way for visitors to get in touch
- **Responsive Design**: Optimized for all device sizes
- **Dark Theme**: Modern aesthetic with ambient lighting effects

## 🛠️ Tech Stack

- **Frontend**: Next.js 16.2.3, React 19.2.4, Tailwind CSS 4
- **Icons**: React Icons
- **Styling**: Tailwind CSS with custom gradients and animations
- **Deployment**: Vercel

## 📦 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/EmonHira135923/Emon-Portfolio.git
   cd emon-portfolio
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🚀 Build for Production

```bash
npm run build
npm start
```

## 📝 Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
